# 📖 摸鱼小说阅读器 loafing-reader
油猴插件，内嵌浏览器里用来上班摸鱼看小说。

+ 油叉地址：[摸鱼小说阅读器 Loafing-Reader](https://greasyfork.org/zh-CN/scripts/470914-%E6%91%B8%E9%B1%BC%E5%B0%8F%E8%AF%B4%E9%98%85%E8%AF%BB%E5%99%A8-loafing-reader)

## 💡 使用说明
| 操作      | 功能     |
| --------- | -------- |
| Shift + R | 唤醒面板 |
| 鼠标移出  | 隐藏面板 |
| 左键      | 翻页     |
| 右键      | 上翻     |

## 🔖 小提示
+ 目前仅支持导入`.txt`格式的文本
+ 浏览器会帮你保存你上次看的小说和位置
